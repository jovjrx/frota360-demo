# Frota360 Demo

Versão de demonstração do Frota360 com dados 100% mockados.

## 🚀 Início Rápido

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## 📊 Dados Disponíveis
